# Blessing of Regrowth

Blessing of Regrowth is a mod that introduces a new item, called Jungle's Blessing, which drops from Plantera. The Jungle's Blessing unlocks a toggle that allows the player to replant trees when chopping them down like the Axe of Regrowth.

## Installation

1. Download the latest release from the [GitHub releases page](https://github.com/ProfBrickz-Terraria-Mods/BlessingOfRegrowth/releases).
2. Place the downloaded `.tmod` file in your `tModLoader/Mods` directory.
3. Launch tModLoader and enable the Blessing of Regrowth mod.
